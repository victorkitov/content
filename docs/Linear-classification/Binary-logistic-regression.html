Бинарная логистическая регрессия

Логистическая регрессия (logistic regression) представляет собой другой популярный метод линейной классификации. По сути это метод линейной классификации с логистической функцией потерь. Достоинством метода является то, что он может выдавать не только метки классов, но и вероятности классов.

Для удобства обозначений включим дополнительный признак, равный тождественной единице в число признаков:

x=[1,x1,x2,...xD]w=[w0,w1,w2,...wD]\begin{align} x&=[1,x^1,x^2,...x^D] \\ w&=[w_0,w_1,w_2,...w_D] \end{align}xw​=[1,x1,x2,...xD]=[w0​,w1​,w2​,...wD​]​​

Тогда линейный бинарный классификатор можно переписать в более компактном виде:

y^=sign(wTx)\hat{y}=sign(w^T x)y^​=sign(wTx)

Эквивалентно логистическая регрессия может быть переформулирована в виде вероятностной модели, выдающей вероятность положителmного класса по правилу:

p(y=+1∣x)=σ(wTx)p(y=+1|x)=\sigma(w^{T}x)p(y=+1∣x)=σ(wTx)

где сигмоида σ(z)\sigma(z)σ(z) представляет собой следующую функцию:

Она удовлетворяет следующему свойству:

1−σ(z)=1−11+e−z=e−z1+e−z=11+ez=σ(−z)1-\sigma(z)=1-\frac{1}{1+e^{-z}}=\frac{e^{-z}}{1+e^{-z}}=\frac{1}{1+e^{z}}=\sigma(-z)1−σ(z)=1−1+e−z1​=1+e−ze−z​=1+ez1​=σ(−z)

поэтому

p(y=−1∣x)=1−p(y=+1∣x)=σ(−wTx)p(y=-1|x)=1-p(y=+1|x)=\sigma(-w^{T}x)p(y=−1∣x)=1−p(y=+1∣x)=σ(−wTx)

Таким образом, для y∈{+1,−1}y\in\{+1,-1\}y∈{+1,−1} вероятностный прогноз строится по правилу:

p(y∣x)=σ(y⟨w,x⟩)p(y|x)=\sigma(y\langle w,x\rangle)p(y∣x)=σ(y⟨w,x⟩)

Оценим w методом условного максимального правдоподобия:

P(Y∣X)=∏n=1Np(yn∣xn)=∏n=1Nσ(⟨w,xn⟩yn)=∏n=1N11+e−⟨w,xn⟩yn→max⁡wP(Y|X)=\prod_{n=1}^{N}p(y_{n}|x_{n})=\prod_{n=1}^{N}\sigma(\langle w,x_{n}\rangle y_{n})=\prod_{n=1}^{N}\frac{1}{1+e^{-\langle w,x_{n}\rangle y_{n}}}\to\max_{w}P(Y∣X)=n=1∏N​p(yn​∣xn​)=n=1∏N​σ(⟨w,xn​⟩yn​)=n=1∏N​1+e−⟨w,xn​⟩yn​1​→wmax​

Максимизация положительной функцию эквивалентна минимизации обратной к ней, поэтому

∏n=1N(1+e−⟨w,xn⟩yn)→min⁡w\prod_{n=1}^{N}\left(1+e^{-\langle w,x_{n}\rangle y_{n}}\right)\to\min_{w}n=1∏N​(1+e−⟨w,xn​⟩yn​)→wmin​

Прологарифмировав критерий, получим классическую задачу минимизации эмпирического риска с логистической функцией потерь (logistic loss):

∑n=1Nlog⁡2(1+e−⟨w,xn⟩yn)→min⁡w\sum_{n=1}^{N}\log_{2}(1+e^{-\langle w,x_{n}\rangle y_{n}})\to\min_{w}n=1∑N​log2​(1+e−⟨w,xn​⟩yn​)→wmin​
