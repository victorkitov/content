Оптимизация по целевым мерам качества

Оптимизация параметров модели​

Меры MAE и WAPE также можно оптимизировать напрямую, но для этого придётся в качестве потерь задавать модули, а не квадраты ошибок и использовать для этого уже численную оптимизацию. Возможность минимизировать модули, а не квадраты ошибок, реализована в большинстве пакетов машинного обучения.

Для минимизации MAPE и MAPE' можно минимизировать модули ошибок, где каждый объект учитывается взвешенным образом с весом 1∣yn∣\frac{1}{|y_n|}∣yn​∣1​ и 1max⁡{∣yn∣,a}\frac{1}{\max\{|y_n|, a\}}max{∣yn​∣,a}1​ соответственно.

Оптимизация гиперпараметров модели​

Гиперпараметры модели также рекомендуется подбирать, оптимизируя целевую меру качества. Сделать это еще проще, чем при настройке параметров (обучении модели), поскольку чаще всего гиперпараметры подбираются перебором. В этом случае достаточно вычислять целевую меру качества, а не меру качества по умолчанию.
