Влияние признаков на качество прогнозов

Метод перестановочной важности признаков (permutation feature importance) представляет собой общий способ расчёта важности каждого признака на прогнозы модели для

  • любой модели (white box, black box models)

  • любой задачи (классификация, регерссия и др.)

  • любой функции потерь

Пусть X∈RN×DX\in\mathbb{R}^{N\times D}X∈RN×D - матрица объекты-признаки (вектора признаков для каждого объекта составляют строки этой матрицы), а Y∈RNY\in\mathbb{R}^{N}Y∈RN - вектор откликов для обучающей (на которой модель настраивалась) или валидационной выборке (не использовавшейся в настройке).

Пусть L(f,X,Y)L\left(f,X,Y\right)L(f,X,Y) - потери модели на выборке X,YX,YX,Y, например, средний модуль ошибки для регрессии или частота ошибок для классификации. Чтобы оценить важность jjj-го признака, перемешаем случайным образом его значения (с возвращением), т.е. значения jjj-го столбца XXX, получим новую матрицу X~j\widetilde{X}_{j}Xj​, отличную от XXX только в jjj-м столбце. При таком случайном перемешивании распределение jjj-го признака сохранится, но связь с yyy потеряется. Пусть L(f,X~j,Y)L\left(f,\widetilde{X}_{j},Y\right)L(f,Xj​,Y) - потери модели на выборке X~j,Y\widetilde{X}_{j},YXj​,Y. Тогда перестановочная важность признака j (permutation feature importance, PMI) считается по одной из следующих формул:

L(f,X~j,Y)L(f,X,Y)(1)\tag{1} \frac{L\left(f,\widetilde{X}_{j},Y\right)}{L\left(f,X,Y\right)}L(f,X,Y)L(f,Xj​,Y)​(1) L(f,X~j,Y)−L(f,X,Y)(2)\tag{2} L\left(f,\widetilde{X}_{j},Y\right)-L\left(f,X,Y\right)L(f,Xj​,Y)−L(f,X,Y)(2)

Рекомендуется использовать отношение (2), а не разность потерь (1), т.к. тогда можно сопоставлять важность признаков на разных задачах с разным уровнем потерь.

Перестановочную важность признаков можно считать

  • по обучающей выборке: тогда узнаем, на какие признаки модель сильнее всего переобучилась

  • по валидационной тестовой выборке: тогда узнаем, какой признак в действительности полезнее для прогнозов.

Методика даёт глобальную интерпретируемость для всей выборки в привязке к конкретной модели и функции потерь. В этом её достоинство по сравнению с внешними эвристическими методами расчёта важности признаков, такими как корреляция с откликом или нормализованная взаимная информация (normalized mutual information, NMI).

К недостаткам методики стоит отнести то, что она может основываться на нереалистичных объектах. Рассмотрим как объекты пациентов больницы, у которых признаки включают рост и вес. Оценивая важность роста, мы переставляем значения роста случайным образом, оставляя вес таким, каким он был, что может приводить к малореальным наблюдениям. Также в случае зависимых признаков если мы переставим случайно значения одного признака, то у модели останется возможность извлекать информацию из оставшихся зависимых признаков. Это приведёт к занижению важности каждого признака в группе зависимых признаков. Поэтому рекомендуется вначале разбивать признаки на группы зависимых, а потом переставлять элементы сразу для всех признаков группы. Так мы оценим важность каждой зависимой группы признаков без смещений.
