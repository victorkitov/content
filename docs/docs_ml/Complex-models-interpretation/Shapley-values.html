Значения Шепли

Значения Шепли позволяют объяснить для каждого объекта x вклад значения каждого признака в прогноз модели f(x)f\left(x\right)f(x). Т.е. оценка производится для конкретного объекта и конкретного значения выбранного признака. Вклад значений всех признаков и обеспечивает результирующий прогноз. Рассмотрим датасет Titanic, представляющий собой описание пассажиров (класс билета, пол, возраст и т.д.) и результаты, выжил ли он в результате затопления Титаника или нет. На рисунке ниже (источник) показано объяснение прогнозируемой вероятности выживания для одного из пассажиров.

Видно, что высокая вероятность прогнозируемого выживания 0.93 была обеспечена тем, пассажир - женщина и не путешествовала третьим классом. Однако то, что она не путешествовала 1м классом (в one-hot кодировании индикаторы класса-разные признаки) несколько снизило оцениваемую вероятность выжить.

Вычисляются значение Шепли для объекта x и вклада значения jjj-го признака усреднением по следующей формуле

ϕj(x)≈1M∑m=1M(f(xm+j)−f(xm−j)),(3)\tag{3} \phi_{j}\left(x\right)\approx\frac{1}{M}\sum_{m=1}^{M}\left(f\left(x_{m}^{+j}\right)-f\left(x_{m}^{-j}\right)\right),ϕj​(x)≈M1​m=1∑M​(f(xm+j​)−f(xm−j​)),(3)

где MMM - число повторов (чем выше, тем точнее оценка), zmz_{m}zm​ - случайный объект для m=1,2,...Mm=1,2,...Mm=1,2,...M, а xm+j,xm−jx_{m}^{+j},x_{m}^{-j}xm+j​,xm−j​ версии xxx, в которых для одинакового случайного подмножества признаков вместо x подставлены значения признаков zmz_{m}zm​, при этом

  • в xm+jx_{m}^{+j}xm+j​ jjj-й признак равен xjx^{j}xj (исходное значение)

  • в xm−jx_{m}^{-j}xm−j​ - jjj-й признак равен zjz^{j}zj (подставлен из zmz_{m}zm​)

В (3) происходит оценка, насколько в среднем вырастает прогноз, если заменить случайное подмножество признаков значениями из другого случайного объекта оставляя и заменяя исходный признак. В итоге получим среднюю прибавку к прогнозу при наличии признака.

Таким образом, значения Шепли оценивают важность признаков в контексте заданного объекта x, их сумма значений = отклонение f(x) от среднего прогноза по всем объектам. Недостатком метода, как и перестановочной важности признаков, является усреднение по малореальным объектам, поскольку в процедуре оценки генерируются синтетические объекты, для которых часть значений признаков взята из одного объекта, а другая часть - из другого, что может приводить к маловероятным комбинациям признаков, особенно когда признаки сильно скоррелированы.

С помощью значений Шепли можно вычислить глобальную каждого признака, как в методе перестановочной важности признаков. Для этого абсолютные значения Шепли для интересуемого признака усредняются по всем объектам выборки:
