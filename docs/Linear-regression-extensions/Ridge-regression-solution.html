Аналитическое решение для гребневой регрессии

В гребневой регрессии вектор весов β\betaβ находится из условия:

L(β)=∑n=1N(xnTβ−yn)2+λβTβ→min⁡βL(\beta)=\sum_{n=1}^{N}\left(x_{n}^{T}\beta-y_{n}\right)^{2}+\lambda\beta^{T}\beta\to\min_{\beta}L(β)=n=1∑N​(xnT​β−yn​)2+λβTβ→βmin​

Поскольку этот критерий выпуклый (докажите!) минимум является глобальным минимумом и находится из условия ∇L(β)=0\nabla L(\beta)=0∇L(β)=0:

2∑n=1Nxn(xnTβ^−yn)+2λβ^=0,2\sum_{n=1}^{N}x_{n}\left(x_{n}^{T}\widehat{\beta}-y_{n}\right)+2\lambda\widehat{\beta}=0,2n=1∑N​xn​(xnT​β​−yn​)+2λβ​=0,

откуда, используя обозначение I∈RD×DI\in\mathbb{R}^{D\times D}I∈RD×D, для единичной матрицы получаем:

(∑n=1NxnTxn)β^+λIβ^=∑n=1Nxnyn\lparen \sum_{n=1}^N x_n^T x_n \rparen \hat{\beta}+\lambda I \hat{\beta} = \sum_{n=1}^N x_n y_n(n=1∑N​xnT​xn​)β^​+λIβ^​=n=1∑N​xn​yn​ (∑n=1NxnTxn+λI)β^=∑n=1Nxnyn\lparen \sum_{n=1}^N x_n^T x_n +\lambda I \rparen \hat{\beta} = \sum_{n=1}^N x_n y_n(n=1∑N​xnT​xn​+λI)β^​=n=1∑N​xn​yn​

Используя обозначения для матрицы объекты-признаки X∈RN×DX\in\mathbb{R}^{N\times D}X∈RN×D и вектора откликов Y∈RNY\in \mathbb{R}^NY∈RN это можно переписать в виде:

(XTX+λI)β^=XTY\left(X^{T}X+\lambda I\right)\widehat{\beta}=X^{T}Y(XTX+λI)β​=XTY

откуда получим итоговый вид аналитического решения:

β^=(XTX+λI)−1XTY\widehat{\beta}=(X^{T}X+\lambda I)^{-1}X^{T}Yβ​=(XTX+λI)−1XTY
Существование решения

Обратим внимание, что матрица XTX+λIX^{T}X+\lambda IXTX+λI положительно-определена, поэтому всегда невырождена для любых λ>0\lambda>0λ>0. Поэтому решение всегда существует, в отличие от линейной регрессии без регуляризации. Интуитивно решение существует, поскольку даже в случае линейной зависимости признаков регуляризация привносит в него однозначность - среди всех решений нужно найти то, которое обладает максимальной простотой (минимизирует ∥β∥22\|\beta\|_2^2∥β∥22​).
